# Contributing to FlowLang

Thanks for your interest in contributing!

## Quick start
1. Fork the repo
2. Install dependencies:
   - `npm install`
3. Run tests:
   - `npm test`
4. Run lint:
   - `npm run lint`
5. Build all packages:
   - `npm run build`

## Development scripts
- `npm run dev` runs TypeScript builds in watch mode (per package)
- `npm test` runs all tests via workspaces

## Guidelines
- Keep changes focused and well-tested.
- Prefer small PRs.
- Use clear commit messages.
- Add/adjust tests when you change behavior.

## Code style
- TypeScript strict mode
- Prettier formatting
- ESLint for linting
